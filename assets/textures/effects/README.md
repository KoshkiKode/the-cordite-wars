# VFX Textures

Particle system textures: explosions, muzzle flashes, smoke, fire, impact marks.

## Format
- **Format**: `.png` with alpha channel, sprite sheets
- **Common atlases**: `explosion_sheet.png` (8×4 frames), `smoke_sheet.png`, `spark_sheet.png`
