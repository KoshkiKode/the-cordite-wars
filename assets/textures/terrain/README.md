# Terrain Textures

Terrain tile textures: ground types, water, transitions, cliff faces.

## Format
- **Format**: `.png` tiles, 512×512 each
- **Types**: Grass, dirt, sand, rock, snow, water, urban concrete, mud
- **Blending**: Terrain uses vertex-color blending in the terrain shader

## Status
Using Kenney prototype grid textures. Final terrain art pending.
