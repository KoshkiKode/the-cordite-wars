# Unit Textures

Texture atlases and individual unit textures for all six factions.

## Format
- **Format**: `.png` (diffuse/albedo), `.png` (normal), `.png` (ORM — Occlusion/Roughness/Metallic)
- **Resolution**: 512×512 per unit (2048×2048 atlas per faction)
- **Naming**: `{faction}_{unit_id}_albedo.png`, `{faction}_{unit_id}_normal.png`, `{faction}_{unit_id}_orm.png`

## Status
Currently using prototype palette textures from Kenney. Final art pending.
