# UI Textures & Icons

Icons for units, buildings, upgrades, resources, and HUD elements.

## Format
- **Unit/building icons**: 64×64 `.png`, `{faction}_{id}_icon.png`
- **UI chrome**: Panel borders, button states, HUD frames
- **Faction emblems**: 256×256 `.png` per faction

## Status
UI is currently fully procedural (no image assets needed). Icons will be added for the command card and minimap overlays.
