# Building Textures

Texture sets for all faction buildings.

## Format
- **Format**: `.png` (albedo, normal, ORM)
- **Resolution**: 1024×1024 per building type
- **Naming**: `{faction}_{building_id}_albedo.png`
- **Variants**: Three damage states (intact/damaged/destroyed) use the same mesh with different textures

## Status
Currently using prototype palette textures. Final art pending.
