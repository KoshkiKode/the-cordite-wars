# Shaders

Godot `.gdshader` / `.gdshaderinc` files for custom visual effects.

## Planned Shaders
- `terrain_blend.gdshader` — Multi-layer terrain texture blending with vertex colors
- `unit_faction_color.gdshader` — Team-color tinting for unit models (uses material parameter)
- `fog_of_war_overlay.gdshader` — Screen-space fog of war mask rendering
- `selection_ring.gdshader` — Animated selection circle under selected units
- `building_hologram.gdshader` — Blue hologram preview when placing buildings
- `cordite_shimmer.gdshader` — Resource node pulsing glow effect
- `explosion_billboard.gdshader` — Camera-facing explosion sprite sheet player

## Status
Currently using Godot's StandardMaterial3D. Custom shaders pending art integration.
