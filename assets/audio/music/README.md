# Music Tracks

Background music for menus, loading screens, and battle states.

## Format
- **Format**: `.ogg` (Ogg Vorbis), loop points set in Godot import settings
- **Bitrate**: 128–192 kbps stereo

## Tracks Needed
| ID | Usage | Length |
|----|-------|--------|
| `music_menu` | Main menu loop | 2–4 min |
| `music_loading` | Loading screen | 30–60 sec |
| `music_battle_calm` | Early game / low intensity | 2–3 min |
| `music_battle_intense` | Mid-late game / high intensity | 2–3 min |
| `music_victory` | Victory screen sting | 30–60 sec |
| `music_defeat` | Defeat screen sting | 20–40 sec |
