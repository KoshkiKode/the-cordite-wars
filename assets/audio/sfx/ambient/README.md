# Ambient Sound Effects

Environmental soundscapes: wind, rain, wildlife, machinery hum, streams.

## Format
- **Format**: `.wav` / `.ogg`, stereo where applicable, designed for seamless looping

## Sources (all CC0)

### nature/ — Natural environment ambience
Sourced from **Kenney Nature Audio** (kenney.nl) — CC0 / Public Domain

- `nature/Ambient/` — Outdoor bird/water ambience, rainy night, storm + thunder
- `nature/Foliage/` — Leaf and foliage rustling sounds
- `nature/Footstep/` — Grass footstep sounds
- `nature/Stream/` — Flowing stream / river with birds
