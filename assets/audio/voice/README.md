# Voice Acting

Unit acknowledgement lines and commander dialogue.

## Format
- **Format**: `.ogg`, mono, 44.1 kHz

## Sources (all CC0)

### Kenney Voiceover Pack #1 (assets/audio/sfx/voice/)
By Kenney (kenney.nl) — CC0 / Public Domain
Male voice: Jeffrey M. Smith / Female voice: Giselle

War-related clips used:
- `war_go_go_go`, `war_cover_me`, `war_watch_my_back` — move acknowledgements
- `war_fire_in_the_hole`, `war_target_engaged`, `war_suppressing_fire`, `war_rpg` — attack acknowledgements
- `war_look_out`, `war_get_down`, `war_call_for_backup` — under-attack alerts
- `war_medic`, `war_sniper`, `war_reloading` — tactical callouts
- `war_target_destroyed` — unit destroyed confirmation
- `war_hold`, `hold`, `ready`, `go` — order acknowledgements
- `mission_completed`, `objective_achieved`, `mission_failed`, `game_over` — mission events
- `congratulations`, `you_win`, `you_lose`, `hurry_up` — game events
