# Fonts

Game UI fonts. Godot 4 uses `.ttf` / `.otf` files loaded via FontFile resources.

## Fonts Planned
- **Primary UI font**: Clean sans-serif for menus and HUD (e.g. Exo 2, Rajdhani, or Orbitron)
- **Monospace/data font**: For resource counts, coordinates, debug overlay
- **Title font**: Bold display font for faction names and mission titles

## Usage
Font files placed here are referenced in `UITheme.cs`. Currently the game uses Godot's built-in default font (Noto Sans).
