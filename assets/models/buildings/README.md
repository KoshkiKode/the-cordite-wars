# Building 3D Models

This directory will contain the final game-ready building 3D models for all six factions.

## Format
- **Format**: `.glb` (GLTF Binary)
- **Polycount target**: 300–2000 triangles depending on building size
- **Naming convention**: `{faction}_{building_id}.glb` (e.g. `bastion_citadel_wall.glb`)
- **Variants**: Each building has three visual states: `_intact`, `_damaged`, `_destroyed`

## Status
Currently using placeholder geometry. See `data/buildings/` for building data definitions and `data/asset_manifest.json` for model assignments.
