# Prop & Scenery Models

Environmental decoration models placed by the prop placer system (`src/Game/World/PropPlacer.cs`). These add visual richness to maps without affecting gameplay.

## Format
- **Format**: `.glb` (GLTF Binary)
- **Types**: Vegetation (trees, bushes), ruins, barrels, crates, fencing, industrial debris
- **Naming**: `prop_{category}_{variant}.glb`
- **Prop manifest**: `data/terrain_manifest.json` references which props are available per biome

## Status
Placeholder farm assets from Quaternius are currently active. These will be replaced with war-themed props.
