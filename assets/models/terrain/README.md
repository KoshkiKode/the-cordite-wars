# Terrain 3D Models

This directory will contain modular terrain tile meshes and terrain feature models (trees, rocks, rubble, water features).

## Format
- **Format**: `.glb` (GLTF Binary)
- **Tile size**: 4×4 world units per terrain tile (matching the grid cell size)
- **Types**: Ground tiles, cliff edges, water edges, transitions, height variations
- **Props/features**: Trees, boulders, ruins, fences — see `assets/models/props/`

## Status
Currently using Godot procedural terrain generation with prototype textures. Final art pending.
