# Buildings — Complete Reference

All buildings in Cordite Wars: Six Fronts, organized by category. For detailed faction context, see individual [Faction pages](Factions).

---

## Universal Building Rules

- All buildings except the HQ (Command Center) must be **constructed by a builder unit** or placed directly (varies by faction)
- Buildings have **prerequisites** — specific other buildings that must already exist before construction is allowed
- **Tech Lab** requires both **Barracks** and **Vehicle Factory** to already be built
- Buildings can be **repaired** while damaged; repair speed varies by faction
- Destroyed buildings can be **rebuilt** from scratch
- The **Fortification Network** (Bastion mechanic) buffs buildings within 8 cells of a Command Hub

## Building Prerequisite Chains

**Arcloft:**
`HQ → Refinery → Barracks → Overwatch Tower`
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab → (Skyport upgrade path)`
`Vehicle Factory + Tech Lab → Shipyard`

**Bastion:**
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab → Command Hub → Aegis Generator`
`Vehicle Factory + Tech Lab → Shipyard`

**Ironmarch:**
`HQ → Forward Assembly (anywhere on map)`
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab`
`Vehicle Factory + Tech Lab → Shipyard`

**Kragmore:**
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab`
`Tech Lab + Vehicle Factory → War Forge → (Mammoth unlocked)`
`Vehicle Factory + Tech Lab → Shipyard`

**Stormrend:**
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab → Storm Capacitor`
`Vehicle Factory + Tech Lab → Shipyard`

**Valkyr:**
`HQ → Refinery → Barracks → Vehicle Factory → Airfield`
`Barracks + Vehicle Factory → Tech Lab`
`Tech Lab + Reactor → Carrier Pad`
`Vehicle Factory + Tech Lab → Carrier Berth (Shipyard)`
