Why this build

A single-developer RTS, made the way the classics were made.

Six asymmetric factions

Every front plays differently — distinct economies, tech trees, and signature units.

Deterministic lockstep

Fixed-point math and a seeded RNG keep LAN matches perfectly in sync across platforms.

No backend servers

Multiplayer is local/LAN only. No accounts, no matchmaking queues, no shutdown risk.

Honest installers

MSI, DMG, DEB, Snap, Flatpak, APK, AAB — packaged the way each platform expects.

Open development

All releases also published on GitHub with full changelog. Reproducible CI builds.

Verifiable by checksum

SHA-256 sums for every artifact so you can confirm the bytes match before you run.

Verify your download

Published checksums sit alongside each release. After downloading, run:

# Linux / macOS
sha256sum -c checksums.txt

# Windows (PowerShell)
Get-FileHash .\CorditeWars-Setup.exe -Algorithm SHA256
